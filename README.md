# Toys-Data-Excersice
This project involved using Power BI to load, clean, and analyze data.

After reviewing the dataset for null values and defining key data types, a relational model was created linking the "sales," "products," "store," and "calendar" tables. Calculations such as cost, price, total orders, revenue, and profit were added to the "sales" table. Finally, the data was visualized in an interactive dashboard.

Este proyecto consistió en utilizar Power BI para cargar, limpiar y analizar datos. Después de revisar el conjunto de datos en busca de valores nulos y definir los tipos de datos clave, se creó un modelo relacional vinculando las tablas de "ventas", "productos", "tiendas" y "calendario". Se realizaron cálculos como costo, precio, pedidos totales, ingresos y ganancias. Finalmente, los datos se visualizaron en un dashboard interactivo.
